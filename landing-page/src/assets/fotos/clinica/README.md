# Fotos — Clínica e Ambiente

Coloque aqui as fotos do ambiente da clínica, recepção, sala de atendimento, etc.

## Nomenclatura sugerida

```
clinica-recepcao.jpg
clinica-sala-01.jpg
clinica-sala-02.jpg
clinica-fachada.jpg
dr-rodrigo-perfil.jpg
dr-rodrigo-atendimento.jpg
```

## Formatos aceitos
- JPG / JPEG (recomendado)
- PNG
- WebP

## Dimensões recomendadas
- **Foto de perfil do Dr.**: proporção 3:4, mínimo 600×800px
- **Fotos de ambiente**: proporção 16:9 ou 4:3, mínimo 1200×675px
- **Tamanho máximo**: 800 KB por arquivo

## Como usar no código
Referencie as imagens como:
```html
<img src="assets/fotos/clinica/dr-rodrigo-perfil.jpg" alt="Dr. Rodrigo" />
```
